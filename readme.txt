Duplicate Posts / Pages & Custom Posts easily in Just single click.

== Description ==

This Plugin is use for Duplicate/Clone Posts / Pages & Custom Posts Types easily in Just single click.

you can Copy Pages/Post/Custom Post Easily with this Plugin


== Installation ==

1. Upload `sm-easy-duplicator` plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. You will find SM Easy Duplicator Plugin in your WordPress admin screen.

== Frequently Asked Questions ==

= Is this plugin Clone Post? 
Yes, this plugin clone the post & page both.
